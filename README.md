# 我的餐廳清單
運用node.js和express進行開發，呈現8家餐廳清單，並具有餐廳詳細選單及搜尋餐廳之功能。

## 安裝環境

+ Windows 8
+ Node.js v10.15.0
+ Express ^4.17.1

## 安裝
1. 下載restaurant_list資料夾，並輸入以下指令下載
```
$ git clone https://github.com/a62262002/restaurant_list.git
```
2. 從終端機安裝npm套件，輸入以下指令
```
$ npm install
```
3. 執行專案
```
$ npm run dev
```
4. 輸入網址並呈現我的餐廳清單介面
```
http://localhost:3000
```
